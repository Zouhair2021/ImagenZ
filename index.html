ImagenZ

Application Java qui génère automatiquement des images pédagogiques à partir d’icônes (animaux, objets…) pour représenter des nombres et des additions.

Outil pour enseignants du primaire

ImagenZ génère simplement des images .png que vous pouvez insérer dans des fiches d’activités, des exercices ou des présentations.

Fonctionnalités principales

  • Représentation automatique de nombres simples et d’additions.
  • Utilisation de catégories d’images (animaux, objets, etc.).
  • Génération de plusieurs captures à la suite.
  • Enregistrement automatique en .png.
  • Gestion intelligente des anciens fichiers (renommage ou suppression).
  • Sauvegarde des préférences utilisateur.

Pour les développeurs

Application développée en Java avec JavaFX. Le dépôt contient l’ensemble du code source ainsi que les ressources utilisées.
